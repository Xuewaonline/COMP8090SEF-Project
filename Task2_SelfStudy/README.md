## How to Run

### Prerequisites
- Python 3.7 or higher
- No external packages required

### Run Graph Demo
```bash
cd Task2_SelfStudy
python graph.py
```

This will demonstrate:
- Undirected Graph: friendship network with BFS/DFS traversal
- Directed Graph: task dependencies with cycle detection and topological sort
- Weighted Graph: city distances with Dijkstra's shortest path

### Run Cocktail Sort Demo
```bash
cd Task2_SelfStudy
python cocktail_sort.py
```

This will demonstrate:
- Step-by-step sorting process
- Best case (already sorted), worst case (reverse sorted), and average case
- Comparison with standard Bubble Sort (passes and swaps count)
- Sorting strings (stable sort property)
